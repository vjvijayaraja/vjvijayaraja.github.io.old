Sep 2024 - Oct 2024

Associated with Laurel Heights Secondary School

About the Project

The Line Follower Robot project involved building and programming a small, autonomous robot capable of following a marked path using sensors and C++ programming. Constructed on a CD base, I designed the schematic for the circuit board using AutoCAD to ensure optimal layout, then created the physical board by carefully arranging and soldering each component, including phototransistors, PIC chips, LEDs, and motor connections. I programmed the PIC chip using C++ to process real-time sensor data, control motor output, and guide the robot's precise adjustments along a line path. My robot worked exceptionally well, and the quality of my soldering and connections earned me a grade of 100 for the project.

Chrome Extension

About the Project

The Gmail AI Spam Filter is a Chrome extension that leverages advanced pattern recognition to provide real-time spam detection directly within Gmail's interface. This powerful tool enhances Gmail's native spam filtering capabilities by implementing an intelligent, probability-based classification system that analyzes email content and patterns to identify potential spam messages. The extension seamlessly integrates with Gmail's UI, offering intuitive visual indicators and probability scores for each email, while maintaining high performance through efficient background processing and real-time updates. Built with modern web technologies and following Chrome's Manifest V3 architecture, this extension demonstrates robust error handling, persistent connection management, and seamless user experience across Gmail sessions.
